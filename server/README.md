# easyVim Node Server

What you need:
  NodeJs
  MongoDB
  .env file

  First time:
    Run in your project root:
      --npm install--

  For build:
    npm start
  For more info:
    DEBUG=server:* npm start
    
To add new dependences: 
  --for tools-- 
    npm install --save [dependency-name]
    npm install --save-dev [dependency-name]
