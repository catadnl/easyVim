# easyVim Node Server

What you need:
  NodeJs
  MongoDB
  .env file

  First time:
    Run in your project root:
      --npm install--

  For build:
    npm start (the server restarts automatically if files are being modified)
    
To add new dependences: 
  --for tools-- 
    npm install --save [dependency-name]
    npm install --save-dev [dependency-name]
